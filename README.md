# VIPER Architectureの本 データ集

## ディレクトリ構成

`errata` : 正誤表を入れておきます。
`sample` : それぞれの章単位でのサンプルソースコードです。
`data` : 本に含まれているデータで、VIPERを使って行く上で利用価値のあるデータです。おまけです。
